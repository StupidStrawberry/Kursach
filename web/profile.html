Как получить API ID Telegram?

Шаг 1: Перейдите на сайт разработчиков Telegram

Откройте my.telegram.org и авторизуйтесь по своему номеру телефона

Шаг 2: Перейдите на создание нового приложения

После авторизации нажмите на "API development tools"

Шаг 3: Создайте новое приложение

В разделе "API development tools" заполните основные данные (можете повторить их с примера)

Шаг 4: Скопируйте нужную информацию

В разделе "App configuration" скопируйте Ваш API ID

Как получить VK токен доступа?

Шаг 1: Перейдите на сайт VK HOST

Откройте vkhost.github.io/ и выберете "VK Admin"

Шаг 2: Разрешите доступ к Вашему аккаунту

После авторизации нажмите "Разрешить" в конце страницы

Шаг 3: Скопируйте Ваш токен

В адресной строке скопируйте Ваш VK Token (он имеет вид vk1.****)
